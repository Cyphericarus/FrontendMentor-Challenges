This project is a **Bento Grid Main Component** layout created as a solution for a challenge from [**Frontend Mentor**](https://www.frontendmentor.io/challenges/bento-grid-RMydElrlOj).

It displays a simple page component of Social Media Management Features in a grid layout based on different screen widths.

* **Grid Layout**: Implementing the grid layout accurately is a bit complex initially.
* **CSS Flexbox**: Adjusting the layouts of flex box for different screen widths.
