This project is a **Fylo Data Storage Component** layout created as a solution for a challenge from [**Frontend Mentor**](https://www.frontendmentor.io/challenges/fylo-data-storage-component-1dZPRbV5n).

It displays a simple page component of Fylo(storage website) based on different screen widths.
