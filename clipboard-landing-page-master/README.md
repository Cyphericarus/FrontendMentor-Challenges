## âš¡ Challenges

* **Grid Layout**: Implementing the grid layout accurately is a bit complex initially.
* **CSS Flexbox**: Adjusting the layouts of flex box for different screen widths.
* **Fixed widths**: Ensuring proper adjustments of widths of the elements.
* **Precise Design**: Implementing the pixel perfect design based on the given designs for all screen widths.
* **Typography**: Adjustiing the sizes, weights, letter-spacing, line-heights of the font is time consuming.
